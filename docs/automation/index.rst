Automation
==========
One of the main objectives the Repository of Organized Curriculums is to enable
the automated dicovery of content correlations: given a curriculum standard document
(a tree of standard nodes) and a collection of learning resources, use machine learning
algorithms to find which content nodes are relevant for each standard node.
